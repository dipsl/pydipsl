Schema highliterÔÉÅ

DIP schema highliter is design to highlight only the most basic concepts of DIP language. Definition of the schema highlighter is in file dipsl/pygments/DIP_Lexer_Schema.py. The following block summarizes all highliter possibilities.

<indent><name> <type> = <value> <unit> # comment

<indent>= <value> <unit>               # comment
<indent>!options <value> <unit>        # comment
<indent>!condition ('<expression>')
<indent>!condition ("<expression>")    # comment
<indent>!condition ("""
<expression>
""")
<indent>!format <value>
<indent>!constant

$source <name> = <path>

{<source>?<query>}
{<source>?<query>}[<slice>]

("<expression>")
('<expression>')
("""
<expression>
""")
